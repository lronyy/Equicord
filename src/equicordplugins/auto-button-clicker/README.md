# Auto Button Clicker Vencord Plugin

I wanted to get ranked in MCTiers but pressing a button to get on the queue is pretty tiring

So i coded this

Have Fun

# Installation

## Prerequisites
- **Node.js 18+** required
- Git installed

## Then just run the following commands:
```sh
git clone https://github.com/Vendicated/Vencord.git
cd Vencord
npm install -g pnpm@10.4.1
pnpm install --frozen-lockfile
git clone https://github.com/S42yt/auto-button-clicker src/plugins/autoButtonClicker
pnpm build
pnpm inject
```
